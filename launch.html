Vibe the Carpet

To launch the game, follow these steps:

  1. Run the local server by opening a command prompt in this folder and typing: node server.js
  2. If you don't have Node.js installed, download and install it first
  3. Once the server is running, click the button below to play the game
Launch Game

Note: The game needs to run on a local server to avoid security restrictions in modern browsers that prevent loading files directly from the filesystem.
